# personal-page

Source for [hugomartins.pm](https://hugomartins.pm) — the personal site of Hugo Martins, Senior Product Manager (Berlin).

## Stack

Next.js (App Router) · TypeScript · Tailwind · Motion · deployed on Vercel.

## Development

```bash
npm install
npm run dev
```

## Structure

```
app/          routes
components/   UI
content/      prototypes (MDX), CV data, site config
public/       media, CV PDF
```
